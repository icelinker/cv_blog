使用coding comment 作为评论系统

使用该插件需要做两个事情： 1. 注册 Coding.net 用户，创建公开项目，创建 oauth 应用。 2. 在 html 中引入 js/css 资源，写初始化代码。

使用此插件需要基础的 html/js 能力

使用流程

按照配置项说明得到一个配置值如下。

coding_comments:
  owner: 'wusisu'
  repo: 'wusisu'
  oauth:
    client_id: '562b3b6b49cadee03464d430c9ccd7c0'
    client_secret: 'bada319fff24ae2e91f2800340cdc0dcfe85fdf7'

在 html 中引用 coding-comments 的资源如下：

<link rel="stylesheet" href="//dn-coding-net-public-file.qbox.me/Coding-Comments/v0.1.1/default.css">
<script type="text/javascript" src="//dn-coding-net-public-file.qbox.me/Coding-Comments/v0.1.1/gitment.min.js"></script>

或者随时跟踪最新发布版本（尽最大努力避免不兼容的升级）

<link rel="stylesheet" href="//dn-coding-net-public-file.qbox.me/Coding-Comments/latest/default.css">
<script type="text/javascript" src="//dn-coding-net-public-file.qbox.me/Coding-Comments/latest/gitment.min.js"></script>

在 html 中写初始化代码如下：

<div id="container"></div>
<script>
    var clientId = '562b3b6b49cadee03464d430c9ccd7c0'
    var clientSecret = 'bada319fff24ae2e91f2800340cdc0dcfe85fdf7'
    var gitment = new Gitment({
      owner: 'wusisu',
      repo: 'wusisu',
      oauth: {
        client_id: clientId,
        client_secret: clientSecret,
      },
    })
    document.getElementById('container').appendChild(gitment.render())
  </script>

即可。

配置项说明

  • owner: Coding.net 的用户名。
  • repo: 你所拥有的一个公开项目的名字。
  • oauth.{client_id & client_secret}: 你注册的 oauth app 的密钥。

注册 oauth app

访问应用管理页，点击添加应用。

填写应用信息，创建应用。注意，出于安全性考虑，此『回调地址』必须为你自己网站的地址。

创建成功后，即可获得你的 client_id 和 client_secret

注意事项

  1. 每个引用 Coding Comments 的页面，默认会使用当前 url 来做为搜索的 key。每个页面的 Coding Comments 需要 owner 在页面上点击初始化。
  2. repo 必须是你所拥有的一个项目。否者无法进行上述初始化的过程。
